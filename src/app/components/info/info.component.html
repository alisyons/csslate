Welcome to CSSlate!

CSSlate is a bachelor thesis project to research how games can impact our learning process. However, this is not the only tool to broaden your understanding of CSS and programming topics. Other helpful resources are:

Flexbox Froggy
Grid Garden
Grasshopper

CSSlate is intended as a extendable open source project, which you can find at Github, so if you want to create your own CSS puzzles, you can simply extend the script. This project is currently in its testing phase.
